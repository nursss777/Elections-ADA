git init - создает git репозиторий из текущей папки (пишется один раз в самом начале)
git add . (либо вместо точки любой файл из дериктории) - добавляет файлы и папки для отслеживания 
в index (эту команду каждый раз перед командой commit)
git commit -m 'cообщение' - создает новую версию если были измемнения в отслеживающихся файлах

полный путь от создания локального репозитория до отправки на удаленный 
1) создаем новый рипозиторий на github 
2) git init - инициализируем локальный репозиторий после этой команды в дериктории должен появиться файл
   .git можно проверить с помошью ls
3) git add. - добавяем все файлы и папки в индекс git
