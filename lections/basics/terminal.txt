Директория это обычная папка 

ctr + alt + t = команда для запуска терминала
sudo = команда для пердоставвления прав суперпользователя 


ls это команда для вывода всех файлов в текущей дериктории 
ls -a команда показывает все скрытые файлы
ls -l - команда выводит инфу детально о файлах и папках

cd test - (change directory) - команда для перехода в другую папку
cd - переход в домошнюю папку
cd.. переходит на 1 папку выше (назад)
cd ../../.. = переход на более высшую папку(несколько папок назад)

mkdir - команда для создания диретории 
mkdir test/test2 = создаст две папки, внутри одной другую
mkdir test test2 = две папки рядом

touch (test.txt)- команда для создания файла 

rm test.py - команда для удаления файла 
rm -rf test.py test - команда для удаления файла и диретории
rmdir <название папки> команда для удаления пустой папки

clear/ctr + l = команда для очистки терминала

mv tset.py test/ - перемещает файл test1.txt в папку test (обязательно указываем / в конце)
mv test.by hello.by переименовывает файл test.by В hello.by
mv test/ test2/ - переименовывает папку test на test2


history - показывает все команды которые были использованы в этом терминале (последние 15 строк)
history 10 - показывает c 10 сторки все команды
history | grep <какое-то слово в скобках> - пказывает все команды с использованием слова в скобках 
    например:
history | grep test5 

pwd - (print working directory) - показывает обсолюный путь до текущей директории (начиная с корневой папки /)

cat <название файла> cat test5

nano test.txt - открывает файл test.txt в редакторе nano, если такого файла нет то создаст его( в случе если не пустой)
control + x, y, Enter или control + s, control + x


code test(code test.txt) - открывает в редакторе кода vs-code указанную папку, или файл
code . - открывает vs-code в текущей дериктории


apt - системный установщик пакетов 
snap - установщик пакетов(но чаще всего установщик приложеий)
